# 채용공고 크롤링 및 중복 제거 파이프라인

SBERT 기반 중복 제거를 활용한 채용공고 수집 및 정제 프로젝트 | 이화여자대학교 정보검색 2026

## 프로젝트 소개
- 채용공고 데이터 14,562건 크롤링
- SBERT 기반 의미론적 중복 제거
- 312건 오류 데이터 수동 검수 및 수정

## 기술 스택
- **Language**: Python
- **Model**: SBERT (Sentence-BERT)
- **Tools**: BeautifulSoup, pandas, sentence-transformers

## 주요 기능
- 채용공고 자동 수집 크롤러
- 문장 임베딩 기반 중복 탐지
- 데이터 정제 및 검수 파이프라인
